# Portfolio

Подборка моих проектов на C++.

- **[Web server](./web-server)** — HTTP/игровой сервер с асинхронным I/O (Boost), PostgreSQL, тестами, Docker/Conan.
- **[Spreadsheet](./spreadsheet)** — мини-Excel: формулы, зависимости ячеек, кэш, ошибки вычислений.
- **[Transport catalogue](./transport-catalogue)** — транспортный справочник: JSON API, маршрутизация, SVG-рендер карты.
- **[Template calculator](./template-calculator)** — шаблонный калькулятор на Qt.

Подробности — в README каждого проекта.
