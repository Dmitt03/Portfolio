\# C++ Spreadsheet



Электронная таблица на C++17 (мини-Excel): поддерживает ячейки с текстом и формулами, ссылки на другие ячейки (A1+B2), вычисление выражений с обработкой ошибок (#REF/#VALUE/#ARITHM), кэширование результатов и защиту от циклических зависимостей. Парсинг формул выполнен через ANTLR (грамматика + AST).



\## Сборка и запуск (Windows, MSVS)

\*Для работы ANTLR вам понадобится Java, проверено на JDK 25 https://www.oracle.com/java/technologies/downloads/


В директории с проектом откройте командную строку и введите следующие команды:

```bat
1. mkdir build
2. cd build
3. cmake .. -DCMAKE\_POLICY\_VERSION\_MINIMUM=3.5
4. cmake --build .
```
